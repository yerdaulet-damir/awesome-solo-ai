# Ideogram MCP — image generation with real typography

> Generate images with legible text — logos, posters, UI mockups — from your agent.

## What is the Ideogram MCP server?

**The Ideogram MCP server lets an AI agent generate images through Ideogram over the Model Context Protocol.** Ideogram's edge is typography: it renders readable, well-placed text inside images, where most image models garble it. The MCP server is hosted with OAuth, so the agent calls it as a native tool.

## Why it matters for solo builders

Most of a solo builder's image needs have words in them — a logo, a launch poster, an OG/social card, a UI mockup. Ideogram is the model that gets the text right, and the MCP makes it a one-line agent request.

## What you can do with it

- Logos and wordmarks with correct lettering.
- Social/OG images and posters with headlines.
- Mockups that include real UI copy.

## Example

> "Make a 1200×630 social card that says 'Awesome Solo AI' in a bold modern style."

## Setup

Connect the hosted Ideogram MCP at [ideogram.ai/features/mcp](https://ideogram.ai/features/mcp) via OAuth in your Claude Code MCP config.
