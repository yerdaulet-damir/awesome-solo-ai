# ElevenLabs MCP — voice, cloning, dubbing, SFX for agents

> Turn text into studio-quality speech, clone a voice, dub video, and generate sound effects from Claude Code.

## What is the ElevenLabs MCP server?

**The ElevenLabs MCP server exposes ElevenLabs' audio API to an AI agent over the Model Context Protocol.** Connected to Claude Code, the agent can synthesize speech, clone a voice, dub video into other languages, and generate sound effects — all as callable tools. It's open-source and maintained by ElevenLabs.

[ElevenLabs](https://elevenlabs.io) is the market-leading TTS and voice platform.

## What you can do with it

- **Text-to-speech** in 70+ languages and thousands of voices.
- **Voice cloning** from a short sample.
- **Dubbing** that preserves the original speaker's voice.
- **Sound effects** from a text description.

## Why it matters for solo builders

Voice is the most expensive-sounding part of content to fake well, and the highest-leverage to automate. With the ElevenLabs MCP, narration, dubbing, and SFX become one agent call inside a content or app pipeline.

## Example

> "Voice this script in my cloned voice, then dub it into Spanish and Portuguese keeping my voice."

## Setup

Get an ElevenLabs API key, then add the [elevenlabs-mcp](https://github.com/elevenlabs/elevenlabs-mcp) server to your Claude Code MCP config.
