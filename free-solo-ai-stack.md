# The Free Solo AI Stack

> Every tool here has a real free tier or is open source — the AI stack you can run as a solo builder for $0 to start.

## What is a free AI stack for solopreneurs?

**A free solo AI stack is the set of AI tools with genuine free tiers or open-source licenses that a one-person business can use to ship video, voice, images, code, and automations at zero starting cost.** This page filters the full [Awesome Solo AI](README.md) list down to only what's free to begin with. Paid upgrades exist, but you can launch without a credit card.

_Last updated: June 2026._

## LLM access (free tier)
- [OpenRouter](https://openrouter.ai) — free models and a free daily request quota across one API. 🆓
- [Groq](https://groq.com) — generous rate-limited free tier, fastest inference. 🆓
- [Together AI](https://together.ai) — signup credit, then cheap open models. 🆓 🔓
- [AI/ML API](https://aimlapi.com) — free-tier multimodal models. 🆓

## Media (free tier)
- [HeyGen](https://heygen.com) — free tier for avatar video. 🆓
- [ElevenLabs](https://elevenlabs.io) — free monthly TTS quota. 🆓
- [Suno](https://suno.com) — free song generations. 🆓
- [Fish Audio](https://fish.audio) — open-source TTS and voice cloning. 🔓
- [OmniVoice Studio](https://github.com/debpalash/OmniVoice-Studio) — fully-local, open-source ElevenLabs alternative; no API keys, 646 languages. 🆓 🔓

## Open-source / open-weight
- [FLUX](https://bfl.ai) — open-weight image models you can self-host. 🔓
- [Cline](https://github.com/cline/cline) — open-source coding agent. 🔓
- [Aider](https://aider.chat) — open-source terminal pair programmer. 🔓
- [n8n](https://n8n.io) — open-source, self-hostable automation. 🔓

## The $0 starting pipeline

A solo builder can run a full content + product loop on free tiers: script with [OpenRouter](https://openrouter.ai), voice with [ElevenLabs](https://elevenlabs.io) free quota, avatar with [HeyGen](https://heygen.com) free tier, build with [Cline](https://github.com/cline/cline) + [ponytail](skills/ponytail.md), and automate with self-hosted [n8n](https://n8n.io). Upgrade only the steps that earn.
