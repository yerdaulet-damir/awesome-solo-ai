# ui-ux-pro-max — design intelligence for Claude Code

> A searchable design knowledge base that turns one plain prompt into a coherent, non-generic design system.

## What is the ui-ux-pro-max skill?

**ui-ux-pro-max is a Claude Code skill that gives the agent a structured design knowledge base, so it produces professional UI without a designer.** From a prompt like "fintech dashboard," a built-in search engine picks a matching style, color palette, font pairing, and layout system — instead of defaulting to the generic look that gives AI apps away.

Created by **nextlevelbuilder** in late 2025, it became one of the most-installed design skills of 2026 (~96,000 GitHub stars).

## What's inside

- **50+ visual styles** (glassmorphism, brutalism, bento, neumorphism, …)
- **161 color palettes** and **57 font pairings**
- **161 product types** and **99 UX guidelines**
- **25 chart types** across **10 stacks** — React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter, Tailwind, shadcn/ui, HTML/CSS

A BM25 search engine matches your prompt to a coherent system, so the choices hang together instead of being random.

## Why it matters for solo builders

The fastest tell of an AI-built product is generic design — Inter font, default blue, no opinion. ui-ux-pro-max gives a solo founder a deliberate, consistent design language across landing page, dashboard, and app, in minutes, with no design hire.

## When to use it

- Building a landing page, dashboard, or app and you want it to *not* look AI-generated.
- You lack a designer and need consistent style decisions.

Pair it with [frontend-design](frontend-design.md) for the strongest anti-slop UI result.

## Install

```
/plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill
/plugin install ui-ux-pro-max@ui-ux-pro-max-skill
```

Then ask Claude Code to design a screen and it will consult the knowledge base.
