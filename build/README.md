## How to use these guides

Each guide is self-contained — start with the one that matches the product you want to ship. Every tool in a guide links to its deep page in this repo (under `tools/`, `mcp/`, or `skills/`) where you can read about configuration, pricing tiers, and alternatives.

If you are not sure where to start: [Build Your Own AI SaaS](./your-own-ai-saas.md) covers the full solo product stack, and [Build Your Own AI Agent](./your-own-ai-agent.md) covers automation — together they account for the majority of things solo builders ship with Claude Code.
