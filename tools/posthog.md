# PostHog — product analytics + LLM observability

> See how users behave and how your AI features perform — cost, latency, quality — in one place.

## What is PostHog?

**PostHog is an open-source product analytics platform with event tracking, session replay, feature flags, A/B testing, and LLM analytics for monitoring AI feature cost, latency, and quality.** For a solo builder it's the single tool that answers both "what are users doing?" and "is my AI feature actually working and affordable?"

## Quick facts

- **Website:** [posthog.com](https://posthog.com)
- **API:** yes — capture + query APIs and SDKs
- **CLI:** SDK/wizard-driven setup
- **MCP:** community MCP servers exist for querying analytics
- **Free tier:** yes — generous monthly free events
- **Open source:** yes — self-hostable

## What you build with it

- Funnels and retention to see what converts (pairs with the [launch playbook](../playbooks/launch-and-get-first-users.md)).
- LLM analytics: track token cost and output quality per feature.
- Feature flags to roll out AI features safely.
