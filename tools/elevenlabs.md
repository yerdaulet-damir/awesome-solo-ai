# ElevenLabs — the voice layer for solo builders

> Market-leading text-to-speech, voice cloning, dubbing, and sound effects.

## What is ElevenLabs?

**ElevenLabs is an AI audio platform for generating studio-quality speech, cloning a voice from a short sample, dubbing video into other languages, and creating sound effects.** It covers 70+ languages and thousands of voices, and exposes everything through an API and an official MCP server, so a solo builder can put production voice into any app or content pipeline.

## Quick facts

- **Website:** [elevenlabs.io](https://elevenlabs.io)
- **API:** yes — full REST API
- **CLI:** community tooling; primary interface is API/SDK
- **MCP:** yes, official and open-source → [ElevenLabs MCP page](../mcp/elevenlabs.md)
- **Free tier:** yes — monthly character quota

## What you build with it

- Voiceovers for [faceless content](../playbooks/faceless-content-factory.md).
- In-app narration, voice agents, and audiobooks.
- Voice-preserving dubbing to reach new language markets.
