# Ideogram — image generation with real typography

> The image model that renders legible, well-placed text — logos, posters, social cards.

## What is Ideogram?

**Ideogram is an AI image generator whose edge is typography: it produces images with readable, correctly-placed text where most models garble it.** It generates and edits via API and ships a hosted MCP server with OAuth, so an agent can request a finished social card or wordmark directly.

## Quick facts

- **Website:** [ideogram.ai](https://ideogram.ai)
- **API:** yes — [developer.ideogram.ai](https://developer.ideogram.ai)
- **CLI:** none official; API-driven
- **MCP:** yes, hosted with OAuth → [Ideogram MCP page](../mcp/ideogram.md)
- **Free tier:** yes — limited generations

## What you build with it

- Logos and wordmarks with correct lettering.
- OG/social images and posters with headlines.
- UI mockups that include real copy.

## Example

> "Make a 1200×630 social card that says 'Awesome Solo AI' in a bold modern style."
