# Nano Banana (Gemini Image) — fast, realistic image gen

> Google's image model: fast, photorealistic, up to 4K, with multi-reference editing.

## What is Nano Banana?

**Nano Banana is Google's Gemini image-generation model, known for fast, realistic output, 4K resolution, and editing from multiple reference images.** It generates and edits images via the Gemini API and tags output with SynthID/C2PA provenance, making it a strong default for solo builders who need quick, believable visuals.

## Quick facts

- **Website:** [gemini.google](https://gemini.google) / [ai.google.dev](https://ai.google.dev)
- **API:** yes — via the Gemini API
- **CLI:** via the `gemini` CLI / Google AI SDKs
- **MCP:** no dedicated server; reachable through Gemini tooling and hosts like [Krea](https://krea.ai)/fal
- **Free tier:** yes — Gemini API free quota

## What you build with it

- Social/OG images, blog heroes, product shots.
- Reference-guided edits (change a scene while keeping a subject).
- Bulk image generation inside an [automation](../README.md#automation--agent-frameworks).
