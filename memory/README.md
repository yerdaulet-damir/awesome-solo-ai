# Agent Memory & Knowledge

> Give your AI agent a persistent brain — so it remembers context beyond the current codebase.

Coding agents forget everything outside the open repo. **Agent memory tools** give a solo builder a queryable, persistent knowledge layer — notes, research, contacts, past decisions — that the agent reads from every session. Each tool has its own page.

## Memory & retrieval layers
- [GBrain](gbrain.md) — synthesis + graph + gap analysis brain by Garry Tan (YC CEO); wire into Claude Code in one command.
- [Context7](../skills/context7.md) — narrower: keeps the agent's *library docs* current.

## Also useful
- [Mem0](https://github.com/mem0ai/mem0) — open-source memory layer for AI agents and apps. 🔓
- [Letta](https://github.com/letta-ai/letta) — stateful agents with long-term memory. 🔓
